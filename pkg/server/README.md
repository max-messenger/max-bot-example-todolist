# Server

Пакет для звпуска серверов http и grpc.

## HTTP

HTTP сервер для обработки запросов, получает `http.Handelr` как зависимость.
Если в конфиге не указан, то сервер не будет запущен.

## System

Системный сервер для обработки запросов информации о приложении и метрик.

По умолчанию включен на порту `8082`.

Можно добавить свои хендлеры через группу `system_handlers`.

По умолчанию включены:

- статус приложения `/health`
- метрики `/metrics`
- информация о приложении `/info`
- профайлинг `/debug`
        - изменение сбора блокировки `/debug/block`
        - изменение сбора мутексов `/debug/mutex`

## GRPC

GRPC сервер для обработки запросов, получает `[]GRPCServiceDefinition` как зависимость.
Если в конфиге не указан, то сервер не будет запущен.
